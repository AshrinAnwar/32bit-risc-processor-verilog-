# 32-Bit RISC Processor using Verilog HDL

A 32-bit RISC processor designed using Verilog HDL and implemented on Basys 3 FPGA using Xilinx Vivado.

---

## Features

- 32-bit RISC Architecture
- Verilog HDL Design
- FPGA Implementation using Basys 3
- Arithmetic and Logical Operations
- Register File
- Control Unit
- Instruction Memory
- Data Memory
- Functional Verification using Testbench

---


## FPGA Resource Utilization

| Resource | Usage |
|---|---|
| LUTs | 25 |
| Slice Registers | 29 |
| Slices | 21 |
| Total Power | 0.124 W |

---

## Project Structure

```text
src/           -> Verilog source code
testbench/     -> Testbench files
constraints/   -> FPGA constraints
images/        -> Output screenshots and diagrams
```

---

## Tools Used

- Verilog HDL
- Xilinx Vivado
- Basys 3 FPGA Board

---

## How to Run

1. Open Vivado
2. Create RTL Project
3. Add Verilog files
4. Add constraints file
5. Run synthesis
6. Run implementation
7. Generate bitstream
8. Program FPGA
